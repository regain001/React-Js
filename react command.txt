1. To create react platform by node package manager
	npm i -g create-react-app@1.5.2

2. create app -
	create-react-app appName

3. run -
	npm start

4. install bootstarp -
	npm i bootstrap@4.1.1

5. Create page router (How to navigate one page to another) -
	npm i create-router-dom

6. for making request (http)- 
	npm i axios

7. for deployment -
	npm run build
